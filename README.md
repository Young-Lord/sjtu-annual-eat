# SJTU-Annual-Eat

把 [Milvoid/SJTU-Annual-Eat](https://github.com/Milvoid/SJTU-Annual-Eat) 使用 TypeScript 重写，可以用于部署到 CloudFlare Worker 上。

## 部署

需要设置 CLIENT_ID 与 CLIENT_SECRET 两个 Secret。
